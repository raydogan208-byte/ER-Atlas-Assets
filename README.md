# ER Atlas Assets

CDN-hosted assets for Elden Ring Atlas, served via jsDelivr.

Base: `https://cdn.jsdelivr.net/gh/raydogan208-byte/ER-Atlas-Assets@v2`

- `icons/elden-ring/items/` — extracted Elden Ring item icons (FromSoftware assets).
- `icons/elden-ring/bosses/` — boss/miniboss portrait screenshots (720×405 webp), one per
  enemy base name; generated by the app repo's `data/src/fetch-boss-images.mjs`.

Releases are immutable jsDelivr tags: push, then tag (`v1`, `v2`, …) and point the app's
`VITE_ASSET_BASE` at the new tag.
